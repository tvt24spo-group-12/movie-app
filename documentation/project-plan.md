# Movie Fans Website - Project Plan

## 1. Overview

This project creates a simple website for movie fans.  
Users can browse movies, create accounts, write reviews, and join groups.

**Tech Stack**

- Frontend: React
- Backend server: Node.js
- Database: PostgreSQL
- Deployment: Deployed as a Docker stack
- API: [The Movie Database](https://api.themoviedb.org/3/)

## 2. Features that need to be implemented

- Responsive design that works on all screen sizes
- User registration with email and password (min 8 chars, 1 capital, 1 number)
- User login and logout
- Delete account and related user data
- Search movies or shows using at least three filters
- View movies currently in Finnish theaters
- Create and view groups
- Send join requests to groups, accept or reject members
- Remove members or leave groups
- Add movies to group pages
- Write movie reviews with text and star rating (1â€“5)
- View all movie reviews
- Create a personal favorite list
- Share favorite list with a public link
- Add one extra custom feature decided by the team

Automated unit tests should include both positive and negative cases for:

- Login
